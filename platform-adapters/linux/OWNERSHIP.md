# Linux platform adapter ownership

**Owner:** Linux platform maintainers.

Owns Linux-specific process, IPC, desktop notification, portal picker, print, credential-prompt, protocol, packaging/updater, sandbox, and session-lifecycle mechanics. The native shell consumes these adapters without gaining product authority.

Tests cover supported distributions and desktop environments, RTL and mixed input surfaces, accessibility integration, secure local IPC, packaging, sandboxing, and process recovery.
