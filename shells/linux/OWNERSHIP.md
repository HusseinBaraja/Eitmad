# Linux shell ownership

**Owner:** Linux UI maintainers.

The Linux shell owns native Linux presentation, accessibility, RTL layout, bidirectional input, navigation, focus, and ephemeral view state. Its UI technology and supported desktop baseline require a separate recorded decision before implementation.

It delegates authoritative work to the Rust engine and OS mechanics to the Linux platform adapter. Tests cover Arabic-first UI behavior, contract conformance, reconnection, localized failures, and forbidden direct authority.
